| 개념            | 설명                          |
| ------------- | --------------------------- |
| **socket**    | 네트워크에서 데이터를 송수신하는 기본 단위     |
| **bind()**    | 서버에서 IP와 포트를 소켓에 연결         |
| **listen()**  | 클라이언트의 접속 요청을 대기 상태로 만듦     |
| **accept()**  | 클라이언트 연결 수락 (새 소켓 반환)       |
| **connect()** | 클라이언트가 서버에 연결 시도            |
| **sendall()** | 데이터를 전송 (bytes 형식)          |
| **recv()**    | 데이터를 수신 (bytes → str 변환 필요) |
| **close()**   | 소켓 닫기 (리소스 반환)              |

## 2번 코드
 : AI 서버와 클라이언트가 TCP/IP로 통신하면서,
클라이언트가 특정 “분석 요청”을 보내면 서버가 그에 맞는 “AI 분석 결과”를 돌려주는 예제
-클라이언트: 요청 조건(mode)과 분석 대상(text)을 JSON 형태로 전송
-서버(AI 서버): 요청을 받아 조건에 따라 분석 수행 후 결과(JSON 형태)로 응답

[클라이언트 입력]
 분석 모드 입력 (length / sentiment / keyword): sentiment
분석할 문장 입력: 오늘은 정말 행복하고 멋진 하루야!

[서버 응답]
: {'result': 'positive', 'desc': '감정 분석 결과: positive'}
